# Study Buddy AI

Study Buddy AI is an AI-powered study assistant designed to help students learn smarter. Users can upload study materials, generate summaries, ask questions from notes, and improve understanding using AI.

## Features

- Upload study notes and PDFs
- AI-generated summaries
- Ask questions from uploaded material
- Clean and modern UI
- Fast and responsive experience

## Tech Stack

- React
- Vite
- Tailwind CSS
- Bun
- Gemini API

## Deployment

Live Demo: https://study-buddy-ai-121.vercel.app

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/study-buddy-ai-121.git

Go into the project folder:

cd study-buddy-ai-121

Install dependencies:

bun install

Run the development server:

bun run dev
