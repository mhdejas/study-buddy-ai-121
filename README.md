Study Buddy AI

Study Buddy AI is an AI-powered learning assistant built to help students study more effectively using artificial intelligence. The application allows users to upload study materials, interact with AI, generate summaries, and improve understanding through smart learning tools.

Features

AI-powered study assistance

Upload and analyze PDFs or notes

Generate quick summaries

Ask questions from uploaded material

Modern and responsive interface

Fast performance with clean UI


Tech Stack

React

Vite

Tailwind CSS

Bun

Gemini API


Live Demo

https://study-buddy-ai-121.vercel.app

Installation

Clone the repository:

git clone https://github.com/mhdejas/study-buddy-ai-121.git  
  
Navigate into the project folder:  
  
cd study-buddy-ai-121  
  
Install dependencies:  
  
bun install  
  
Start the development server:  
  
bun run dev
